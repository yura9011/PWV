# AOE Spell System - Milestone Summary (SIMPLIFICADO)

## Overview
Milestone simplificado para implementar 3 features básicas de AOE sin modificar sistemas existentes:

1. **Ground-Targeted AOE**: Selección básica de área en suelo con mouse
2. **Player-Centered AOE**: Área circular simple alrededor del jugador  
3. **Cone Attacks**: Cono frontal básico del jugador

## Enfoque Minimalista
- **Solo scripts de testing independientes**
- **Visuales básicos** (LineRenderer/primitivas)
- **Sin integración compleja** con AbilitySystem
- **Sin audio, efectos avanzados, o UI compleja**
- **Testing básico** con consola y enemigos dummy

## Timeline Reducido
**Total**: 14 horas (3 días)
- Fase 1: Detección Básica (4h)
- Fase 2: Ground Targeting (3h) 
- Fase 3: Player-Centered (2h)
- Fase 4: Cone Attacks (3h)
- Fase 5: Testing Mínimo (2h)

## Entregables Mínimos
- 4 scripts básicos de testing
- Escena de prueba simple
- Validación por consola
- **NO** integración con sistemas existentes

## Success Criteria Básicos
- Las 3 features detectan targets correctamente
- Indicadores visuales simples funcionan
- Testing básico valida funcionalidad
- **NO** se requiere polish o integración compleja

Enfoque: **Proof of concept básico** de las 3 mecánicas AOE.
